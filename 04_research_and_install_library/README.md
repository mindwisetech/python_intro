# Project 4: Research and Install a Library (Requests)

In this project, you will:

- Research the `requests` library online
- Install it using pip
- Write a script that fetches data from a public API (`https://api.github.com`)
- Display basic information from the fetched JSON data

## Objectives

- Learn how to research Python libraries and their documentation
- Practice installing libraries using pip
- Make HTTP GET requests
- Parse JSON responses

## How to run

1. Install the `requests` library:
```bash
pip install requests
```
2. Run the script:
```bash
python main.py
```
3. The script will fetch and print basic info from the GitHub API.
